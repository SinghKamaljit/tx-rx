About TX_RX

Simple file transfer. No frills.

How it works

Upload any file and receive a 16-character Transfer ID. Share that ID with anyone. They enter it here to download the file. No account, no sign-up, no expiry enforced automatically.

What can I send?

Any file type — documents, PDFs, images, videos, audio, ZIPs, APKs. Any size. TX_RX does not restrict file type or impose a size cap.

Storage

Files are stored on Cloudinary. The Transfer ID lookup is handled by Firebase Firestore. No personal data is collected.

Security

  • Randomised 16-character Transfer IDs
  • Files not indexed or searchable
  • Only accessible via exact Transfer ID
  • Anonymous Firebase Auth protects Firestore writes

TX_RX

TX = Transmit. RX = Receive. A standalone PWA. Independent of all other apps.
